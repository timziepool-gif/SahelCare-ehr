## 📖 Overview

The **Sahel Family Health Clinic (SFHC) Electronic Health Record System** is a modern, end-to-end Hospital Information System (HIS) designed to streamline clinical and administrative workflows within healthcare facilities.

This application provides a fully integrated platform covering patient management, appointments, clinical documentation, laboratory services, pharmacy operations, billing, inventory, administration, telemedicine, reporting, and patient engagement.

The system was developed as a comprehensive healthcare informatics project using modern web technologies and follows real-world healthcare workflows.
